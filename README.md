# JavaScript Array Operations

A simple JavaScript program that performs different operations on an array using functions.

## Objective

The objective of this assignment is to create JavaScript functions that receive an array as an argument and perform various operations such as finding the maximum number, calculating the sum of all elements, and counting the number of odd numbers.

## Technologies Used

- HTML5
- JavaScript

## Project Structure

```text
JavaScript-Array-Operations/
│── index.html
│── script.js
└── README.md
```

## Features

- Find the maximum number in an array.
- Calculate the sum of all array elements.
- Count the number of odd elements.
- Demonstrates the use of:
  - Arrow Function
  - Anonymous Function
  - Normal Function
- Display results using `console.log()`.

## How to Run

1. Clone or download the project. https://github.com/kjwashim/Task-15-JavaScript-Arrays-and-Functions/tree/main
2. Open the project folder.
3. Open `index.html` in your browser.
4. Press **F12** and open the **Console** tab.

## Sample Output

```text
Ex Array: [4, 8, 2, 11, 6, 7, 10]
Maximum Number: 11
Sum of All Elements: 48
Count of All Odd Numbers: 2
```

## Learning Outcomes

- Practice working with arrays.
- Learn different types of JavaScript functions.
- Iterate through arrays using loops.
- Use conditional statements.
- Display output using `console.log()`.
